## 詳細設計の進め方

基本設計の一覧から以下の個別設計書を作成します：

```
docs/
├── api/
│   ├── api-list.md          ← 読み込み済み（一覧）
│   ├── API-001.md           ← 個別エンドポイント詳細
│   ├── API-002.md
│   └── ...
├── screens/
│   ├── screen-list.md       ← 読み込み済み（一覧）
│   ├── SC-001.md            ← 個別画面詳細
│   ├── SC-002.md
│   └── ...
└── database/
    ├── er-design.md          ← 読み込み済み（全体像）
    └── TBL-001.md            ← 必要に応じて個別テーブル詳細
```
